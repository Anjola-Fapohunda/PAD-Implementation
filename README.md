# PAD-Implementation
An implementation of the Patch Agnostic Defense (PAD) on MITRE's APRICOT dataset. The GitHub for the paper can be found here: https://github.com/rupy/HumanDetection.git and the paper is linked here: https://arxiv.org/pdf/2404.16452.

The paper's repository does not provide the dataset it runs on, but a closer look at the code tells us it is the INRIA dataset. It is publicly available online.

## Dependencies
The requirements for running this code are:
1. Python 3.8
2. Meta's Segment Anything project. You can install it using this code: wget https://dl.fbaipublicfiles.com/segment_anything/sam_vit_l_0b3195.pth
3. See requirements.txt for additional information
